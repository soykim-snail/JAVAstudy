[방법 2 : Scanner]

or `java.util.Scanner`!! very easy!  (from **Java 5**(JDK 1.5))





```java
import java.util.Scanner;
class ScannerTest {
    public static void main(String[] args){
        Scanner sc = new Scanner(System.in);
        int word = sc.next();
        sc.close();
    }
}
```
